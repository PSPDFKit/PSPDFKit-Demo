Overview

Returns a lighter and a darker color to make a gradient out of ‘color’.

  • – initWithCGPDFArray:

    Given an array of floats, applies the rules described in the PDF 1.7 Reference (page 607) to derive a UIColor instance. nil argument implies clear/transparent color. returns nil if no color could be derived.
