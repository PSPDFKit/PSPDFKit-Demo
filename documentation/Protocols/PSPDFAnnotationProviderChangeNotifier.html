PSPDFAnnotationProviderChangeNotifier Protocol Reference

Conforms to NSObject
Declared in PSPDFAnnotationProvider.h

Overview

To be notified on any changes PSPDFKit does on your annotations, implement this notifier. It will be set as soon as your class is added to the annotationParser.
