getPixelAtPoint:

Returns a pixel at a given location.

- (PSPDFPixel)getPixelAtPoint:(BMPoint)point

Parameters

point

The point from which a pixel will be taken. For all points in a BitmapContextRep, the x and y values start at 0 and end at width – 1 and height – 1 respectively.

Return Value

The pixel with values taken from the specified point.

setBrightness:

Darken or brighten the image.

- (void)setBrightness:(CGFloat)brightness

Parameters

brightness

A percentage from 0 to 2. In this case, 0 is the darkest and 2 is the brightest. If this is 1, no change will be made.

setPixel:atPoint:

Sets a pixel at a specific location.

- (void)setPixel:(PSPDFPixel)pixel atPoint:(BMPoint)point

Parameters

pixel

An RGBA pixel represented by an array of four floats. Each component is one float long, and goes from 0 to 1. In this case, 0 is black and 1 is white.

point

The location of the pixel to change. For all points in a BitmapContextRep, the x and y values start at 0 and end at width – 1 and height – 1 respectively.
