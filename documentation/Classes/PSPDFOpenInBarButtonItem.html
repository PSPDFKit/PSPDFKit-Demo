Conforms to UIDocumentInteractionControllerDelegate
Declared in PSPDFOpenInBarButtonItem.h

Overview

Open in is only possible if the PSPDFDocument is backed by exactly one file-based PDF. Before sending it to another application, annotations will be saved.
