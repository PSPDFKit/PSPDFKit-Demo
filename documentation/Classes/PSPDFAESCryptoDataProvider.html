Inherits from NSObject

Overview

This class allows a tranparent decryption of AES256 encrypted files. The IV vector for the file is written in the first 16 bytes of the file to read. Use the provided encryption tool to prepare your documents.

Ensure your passphrase/salt are also protected within the binary, or at least obfuscated.

Encryption marginally slows down rendering, since everything is decrypted on the fly. Only available in PSPDFKit Annotate or Source.
