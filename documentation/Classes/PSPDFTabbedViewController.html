PSPDFTabbedViewController Class Reference

Inherits from PSPDFBaseViewController : UIViewController
Declared in PSPDFTabbedViewController.h

Overview

Allows displaying multiple PSPDFDocument’s, easily switchable via a top tab bar.

Tasks

  • – initWithPDFViewController:

    Initialize the controller. Set a custom pdfViewController to use a subclass. If nil, a default instance will be created.
  •   visibleDocument

    Currently visible document.

         property
  •   documents

    Documents that are currently loaded.

         property
  • – addDocuments:atIndex:animated:

    Add one or multiple documents to the documents array at the specified index. Documents that are already within documents (or are equal to those) are ignored. If index is invalid or NSUIntegerMax they’re appended at the end. Note: animated is a placeholder, doesn’t yet animate.
  • – removeDocuments:animated:

    Removes one or multiple documents to the documents array. Documents that are not in the documents array will be ignored. Note: animated is a placeholder, doesn’t yet animate.
  •   delegate

    Delegate to capture events.

         property
  •   enableAutomaticStatePersistance

    Set to YES to enable automatic state persisting. Will be saved to NSUserDefaults. Defaults to NO.

         property
  • – persistState

    Persists the state to NSUserDefaults.
  • – restoreState

    Restores state from NSUserDefaults. Returns YES on success. Will set the visibleDocument that is saved in the state.
  • – restoreStateAndMergeWithDocuments:

    Restores the state and merges with new documents. First document in the array will be visibleDocument.
  •   statePersistanceKey

    Defaults to kPSPDFTabbedDocumentsPersistKey. Change if you use multiple instances of PSPDFTabbedViewController.
