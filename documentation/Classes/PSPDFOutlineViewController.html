maximumNumberOfLines

How many lines should be displayed for a cell. Defaults to 4.

@property (nonatomic, assign) NSUInteger maximumNumberOfLines

Discussion

Set this to 1 for PSPDFKit v1 behavior (tail trunication, one line) Set to 0 to show the full text, no matter how long the entry is.
