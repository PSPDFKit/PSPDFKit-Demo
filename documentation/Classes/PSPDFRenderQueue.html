PSPDFRenderQueue Class Reference

Inherits from NSObject
Declared in PSPDFRenderQueue.h

Overview

Render Queue. Does not cache. Used for rendering pages/page parts in PSPDFPageView.

Tasks

  • + sharedRenderQueue

    Render Queue is a singleton.
  • – requestRenderedImageForDocument:forPage:withSize:clippedToRect:withAnnotations:options:delegate:

    Requests a (freshly) rendered image from a specified document. Does not use the file cache. For options, see PSPDFPageRender.
  • – cancelRenderingForDelegate:async:

    Cancels all queued render-calls. Async will perform on the next thread. (don’t use async in dealloc)
  • – hasRenderJobsForDelegate:

    Returns YES if currently a RenderJob is scheduled or running for delegate.
  • – currentRenderJob

    Returns the currently rendered renderJob.
  • – numberOfQueuedJobs

    Return how many jobs are currently queued.
