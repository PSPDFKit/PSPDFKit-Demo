PSPDFDocumentProvider Class Reference

Inherits from NSObject
Declared in PSPDFDocumentProvider.h

Overview

Encapsulates the CoreGraphics-calls to the CGPDF* data providers. We need to keep around the document when accessing a CGPDFPage. Retaining the page alone is not enough.
