PSPDFDocumentProvider Class Reference

Inherits from NSObject
Declared in PSPDFDocumentProvider.h

Overview

A PSPDFDocument consists of one or multiple PSPDFDocumentProvider’s. Note: This class is used within PSPDFDocument and should not be instantiated externally.
